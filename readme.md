# vPlayer
 - 轻量级播放器，支持3种显示模式(最小化、卡片式、全屏)
 - 支持歌词(LRC)，同时**注重性能**，500ms一次刷新
 - **自适应**，不占地，支持简单的**CUE解析**(允许多个CUE)
 - 不支持跨域，请LRC和CUE都下载至自己的服务器

## 用途
对于文件管理器(类似我的项目vList)，**前端解析CUE**还是很不错的
